# What is this?

Easily add shadows.

# Installation

`npm i shadowwizard --save`

Then:
```
import {shadowwizard} from 'shadowwizard';

shadowwizard({
    shadow_type: 'soft',
    padding: false
});
```
## Options

Shadowwizard supports 2 options, both optional.
* *shadow_type* - _hard | soft_ (Defaults to soft)
* *padding* - booLean_ (Defaults to false)
